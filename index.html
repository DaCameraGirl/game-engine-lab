// 03 — quickstart

Get Running Fast

Start with Godot + Panda3D. That covers two categories on the form with skills you already have.

~/GameEngineLab $ pip install Panda3D
Successfully installed Panda3D-1.10.x

~/GameEngineLab $ python demos/panda3d/spinning_scene.py
★ Panda3D window opened — spinning 3D model loaded

~/GameEngineLab $ godot --path demos/godot/ --export-release "HTML5" build/index.html
★ Godot project exported to web — open in browser

~/GameEngineLab $ python launcher.py
★ Desktop launcher started — all demos accessible
